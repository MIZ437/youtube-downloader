================================================================================
                      YouTube Downloader v1.0
                           取扱説明書
================================================================================

【概要】
YouTube動画・Xスペースのダウンロードと、AI文字起こし機能を備えたツールです。
私的利用専用です。著作権法を遵守してご使用ください。

================================================================================
                           使い方
================================================================================

【起動方法】

  1. 「YouTubeDownloader.vbs」をダブルクリック

  2. 初回起動時は自動的に必要なライブラリがインストールされます
     （インターネット接続が必要・数分かかる場合があります）

  3. セットアップ完了後、アプリが自動で起動します

  ※ 2回目以降は数秒で起動します

--------------------------------------------------------------------------------

【事前準備（必須）】

  ● Python 3.9以上がインストールされていること

    インストールされていない場合:
    https://www.python.org/downloads/

    インストール時の注意:
    - 「Add Python to PATH」にチェックを入れてください
    - 「Install for all users」を推奨

--------------------------------------------------------------------------------

【機能説明】

  ■ YouTubeダウンロード
    - 単体動画のダウンロード
    - 複数URL一括ダウンロード（改行区切り）
    - 画質選択（最高画質/1080p/720p/480p/360p）
    - 音声のみ（MP3）ダウンロード
    - 字幕ダウンロード
    - BAN対策機能（ダウンロード間に遅延）

  ■ YouTube再生リスト一括ダウンロード
    - 再生リスト全体のダウンロード
    - フィルター機能（日付/再生回数/再生時間/タイトル）
    - 選択ダウンロード

  ■ Xスペースダウンロード
    - スペースURLからダウンロード
    - ツイートに埋め込まれたスペースにも対応
    - 複数URL一括ダウンロード

  ■ 文字起こし
    - YouTube動画の文字起こし
    - ローカルファイルの文字起こし
    - 3種類のエンジン選択:
      * 標準Whisper（安定）
      * Faster Whisper（2-4倍高速）
      * kotoba-whisper（日本語特化・高精度）
    - カスタム辞書機能（固有名詞の認識向上）
    - 出力形式: タイムスタンプ付き/SRT/プレーンテキスト

--------------------------------------------------------------------------------

【設定】

  メニュー > ファイル > 設定

  ● 保存先
    ダウンロードしたファイルの保存先フォルダ
    ※実際の保存先にはタイムスタンプ付きフォルダが作成されます

  ● ダウンロード設定
    - デフォルト画質
    - 字幕自動ダウンロード

  ● 文字起こし設定
    - Whisperモデル（tiny/base/small/medium/large）
    - デフォルト言語
    - YouTube字幕優先

  ● 精度向上設定
    - Whisperエンジン選択
    - カスタム辞書（固有名詞を登録して認識精度向上）

--------------------------------------------------------------------------------

【カスタム辞書の使い方】

  文字起こしで固有名詞や専門用語がうまく認識されない場合、
  カスタム辞書に登録すると精度が向上します。

  入力例:
    YouTube、ダウンローダー、Whisper、yt-dlp、固有名詞

  ポイント:
    - 読点（、）またはカンマ（,）で区切る
    - 150文字以内が効果的
    - 設定画面で登録すると毎回自動適用
    - 文字起こしタブで一時的に追加も可能

================================================================================
                        トラブルシューティング
================================================================================

【起動しない場合】

  1. Pythonがインストールされているか確認
     - コマンドプロンプトで python --version を実行
     - バージョンが表示されればOK

  2. Pythonを再インストール
     - 「Add Python to PATH」にチェックを入れる

  3. セキュリティソフトの確認
     - .vbsファイルがブロックされている可能性
     - 一時的に除外設定を追加

--------------------------------------------------------------------------------

【ダウンロードエラーの場合】

  1. yt-dlpを更新
     メニュー > ヘルプ > yt-dlpを更新

  2. URLが正しいか確認
     - 非公開/削除された動画ではないか
     - 地域制限がないか

  3. インターネット接続を確認

--------------------------------------------------------------------------------

【文字起こしが遅い場合】

  1. Whisperモデルを小さくする
     - large → medium → small → base → tiny
     - 小さいほど高速（精度は下がる）

  2. Faster Whisperを使用
     - 設定 > 精度向上設定 > エンジン で変更

  3. GPU（NVIDIA）があれば自動で高速化されます

--------------------------------------------------------------------------------

【FFmpegエラーの場合】

  初回起動時に自動ダウンロードされます。
  失敗した場合は手動でダウンロード:

  1. https://github.com/BtbN/FFmpeg-Builds/releases から
     ffmpeg-master-latest-win64-gpl.zip をダウンロード

  2. 解凍してbinフォルダの中身を、
     アプリのffmpegフォルダにコピー
