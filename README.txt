================================================================================
                        YouTube Downloader 説明書
================================================================================

【概要】
YouTube動画のダウンロードと文字起こしができるツールです。
私的利用専用です。著作権法を遵守してご利用ください。

================================================================================
【機能一覧】
================================================================================

1. 動画ダウンロード
   - 単一URL / 複数URL / 再生リストに対応
   - 画質選択（最高画質〜360p、音声のみ）
   - 字幕ダウンロード対応

2. 再生リストフィルタリング
   - アップロード日で絞り込み
   - 再生回数で絞り込み
   - 再生時間で絞り込み
   - タイトルに含む/含まない文字列で絞り込み

3. 文字起こし
   - YouTube字幕取得（高速・優先）
   - Whisper AI音声認識（字幕がない動画用）
   - SRT/TXT形式で保存可能

================================================================================
【使い方】
================================================================================

■ 初回起動
  1. install.bat をダブルクリック（依存パッケージをインストール）
  2. run.bat をダブルクリック（アプリ起動）
  3. 初回はFFmpegの自動セットアップが行われます

■ 動画ダウンロード
  1. 「ダウンロード」タブでURLを入力（複数の場合は改行区切り）
  2. 画質を選択
  3. 「ダウンロード開始」をクリック

■ 再生リストダウンロード
  1. 「再生リスト」タブで再生リストURLを入力
  2. 「読み込み」をクリック
  3. 必要に応じてフィルターを設定
  4. ダウンロードしたい動画にチェック
  5. 「選択した動画をダウンロード」をクリック

■ 文字起こし
  1. 「文字起こし」タブでURLまたはファイルを指定
  2. 言語とモデルを選択
  3. 「文字起こし開始」をクリック
  4. 完了後「結果を保存」で保存

================================================================================
【動作環境・推奨スペック】
================================================================================

■ 必須環境
  - Windows 10 / 11 (64bit)
  - Python 3.10以上（install.batで自動確認）
  - インターネット接続

■ ダウンロード機能のみ使用する場合
  ---------------------------------------------------------------------
  |  レベル  |  CPU           |  メモリ  |  備考                      |
  ---------------------------------------------------------------------
  |  快適    |  任意          |  4GB以上 |  問題なく動作              |
  |  普通    |  任意          |  2GB以上 |  問題なく動作              |
  |  最低    |  任意          |  1GB以上 |  動作可能                  |
  ---------------------------------------------------------------------
  ※ダウンロード機能はスペックをほとんど必要としません

■ 文字起こし機能を使用する場合（重要）
  ---------------------------------------------------------------------
  文字起こしには「Whisper」というAI音声認識を使用します。
  GPUがあると高速ですが、CPUのみでも動作します。

  ★ GPU/CPUは自動判断されます（設定不要）
    - NVIDIA GPU + CUDA がインストール済み → 自動でGPU使用
    - それ以外 → 自動でCPU使用
  ---------------------------------------------------------------------

  【モデル別の要件】

  ┌─────────┬────────────┬────────────┬───────────┬──────────┬─────────────────┐
  │ モデル  │ 必要VRAM   │ 必要RAM    │ 相対速度  │ 精度     │ 推奨用途        │
  │         │ (GPU使用時)│ (CPU使用時)│           │          │                 │
  ├─────────┼────────────┼────────────┼───────────┼──────────┼─────────────────┤
  │ tiny    │ 1GB        │ 2GB        │ 最速(32x) │ 低       │ テスト・短い動画│
  │ base    │ 1GB        │ 2GB        │ 高速(16x) │ 中       │ 一般用途（推奨）│
  │ small   │ 2GB        │ 4GB        │ 中速(6x)  │ 中高     │ バランス重視    │
  │ medium  │ 5GB        │ 8GB        │ 低速(2x)  │ 高       │ 精度重視        │
  │ large   │ 10GB       │ 16GB       │ 最遅(1x)  │ 最高     │ 最高精度が必要  │
  └─────────┴────────────┴────────────┴───────────┴──────────┴─────────────────┘

  【グラフィックボード(GPU)別の目安】

  ┌───────────────────────┬────────┬─────────────────────────────────┐
  │ GPU                   │ VRAM   │ 使用可能モデル                  │
  ├───────────────────────┼────────┼─────────────────────────────────┤
  │ RTX 4090              │ 24GB   │ 全モデル快適                    │
  │ RTX 4080/4070 Ti      │ 16GB   │ 全モデル快適                    │
  │ RTX 4070/3080         │ 12GB   │ large まで快適                  │
  │ RTX 3070/4060 Ti      │ 8GB    │ medium まで快適                 │
  │ RTX 3060/4060         │ 8GB    │ medium まで快適                 │
  │ RTX 3060 (12GB版)     │ 12GB   │ large まで快適                  │
  │ GTX 1660/RTX 2060     │ 6GB    │ small まで快適                  │
  │ GTX 1650/1050 Ti      │ 4GB    │ base まで / small ギリギリ      │
  │ GPU なし (CPU)        │ -      │ tiny〜base 推奨（時間かかる）   │
  └───────────────────────┴────────┴─────────────────────────────────┘

  【スペック別の目安】（10分の動画を文字起こしする場合）

  ◆ 快適に動かせる（1〜3分で完了）
    - NVIDIA GPU: RTX 3060以上（VRAM 8GB以上）
    - システムメモリ: 16GB以上
    - 使用可能モデル: small〜large

  ◆ 普通に動かせる（5〜15分で完了）
    - NVIDIA GPU: GTX 1660以上（VRAM 6GB以上）
    - または 高性能CPU (Core i7/Ryzen 7以上)
    - システムメモリ: 16GB以上
    - 使用可能モデル: base〜small

  ◆ 何とか動かせる（30分〜1時間で完了）
    - CPU のみ (Core i5/Ryzen 5程度)
    - システムメモリ: 8GB以上
    - 使用可能モデル: tiny〜base

  ◆ 動作困難・非推奨
    - システムメモリ 4GB以下
    - 10年以上前のCPU
    - 32bit OS

  【GPU高速化について】
  NVIDIA GPUをお持ちの場合、CUDAを使用すると大幅に高速化されます。
  ※AMD GPU / Intel GPUは現時点で非対応

  CUDAの確認方法:
    1. コマンドプロンプトで「nvidia-smi」を実行
    2. 表示されればCUDA対応GPU搭載

【文字起こしのコツ】
================================================================================

1. まずYouTube字幕を試す
   - 「YouTube字幕を優先」にチェックを入れる
   - 字幕があれば数秒で完了（Whisper不要）

2. Whisperを使う場合
   - 最初は「base」モデルで試す
   - 精度が足りなければ「small」「medium」に上げる
   - GPUがない場合は「tiny」「base」推奨

3. 長い動画の場合
   - 1時間以上の動画はかなり時間がかかります
   - GPUなしの場合、動画の2〜5倍の時間が目安

================================================================================
【トラブルシューティング】
================================================================================

■ ダウンロードが失敗する
  → FFmpegが正しくインストールされているか確認
  → アプリ再起動でFFmpegセットアップダイアログが出るか確認
  → インターネット接続を確認

■ 文字起こしが遅い
  → より小さいモデル（tiny/base）を選択
  → YouTube字幕を優先にチェック

■ メモリ不足エラー
  → より小さいモデルを選択
  → 他のアプリを終了してメモリを確保

■ アプリが起動しない
  → install.bat を再実行
  → Python 3.10以上がインストールされているか確認
